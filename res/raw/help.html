Welcome!

The goal of the game is to guess a 4 color secret code chosen by the computer.

After each guess, the computer will give you a hint to tell you how close you are to the correct code.
The hint will contain a red peg for each correct color AND position, and a white peg for each correct color but incorrect position.

For instance: if the secret code is:
PURPLE GREEN YELLOW ORANGE,
and you guessed:
PURPLE YELLOW RED RED,
the computer will give you:
  • one red peg for PURPLE (correct color AND position)
  • one white peg for YELLOW (color present in the code, but not at the correct position).
Note: the order of the hint pegs is not important!

Now that you have this hint, you can try again to guess the secret code.

If you find it in less than 12 guesses, you have won the game!

How to play

You start with the top, highlighted row.
Either tap (or click with the trackball) on a hole and then pick a color, or use the picker on top of the board to drag and drop to a hole.
When you have chosen the 4 colors of your guess, press the OK button.


Good luck, code breaker!
